## –°–æ–∑–¥–∞–Ω–∏–µ —Å–ø–∏—Å–∫–æ–≤

–°–ø–∏—Å–æ–∫ - —ç—Ç–æ —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—â–∏–π —Å–æ–±–æ–π —É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω—É—é —Å–æ–≤–æ–∫—É–ø–Ω–æ—Å—Ç—å –æ–±—ä–µ–∫—Ç–æ–≤ –¥–∞–Ω–Ω—ã—Ö, —Ç–æ –µ—Å—Ç—å –∏—Ö –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å (sequence).

–ß—Ç–æ–±—ã —Å–æ–∑–¥–∞—Ç—å —Å–ø–∏—Å–æ–∫, –º–æ–∂–Ω–æ –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è —Ñ—É–Ω–∫—Ü–∏–µ–π **list**:

```scheme
(list 1 2 3 4 5)
```

–ó–Ω–∞—á–µ–Ω–∏–µ–º —ç—Ç–æ–≥–æ –≤—ã—Ä–∞–∂–µ–Ω–∏—è —è–≤–ª—è–µ—Ç—Å—è —Å–ø–∏—Å–æ–∫ *'(1 2 3 4 5)*.  
–û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ –Ω–∞ —Ç–æ, —á—Ç–æ –ø–µ—Ä–µ–¥ –æ—Ç–∫—Ä—ã–≤–∞—é—â–µ–π —Å–∫–æ–±–∫–æ–π –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –æ–¥–∏–Ω–∞—Ä–Ω–∞—è –∫–∞–≤—ã—á–∫–∞ (–∞–ø–æ—Å—Ç—Ä–æ—Ñ). –≠—Ç–æ —Ü–∏—Ç–∞—Ç–∞ - –æ—Å–æ–±–∞—è —Å–∏–Ω—Ç–∞–∫—Å–∏—á–µ—Å–∫–∞—è —Ñ–æ—Ä–º–∞, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–¥–∞–≤–ª—è–µ—Ç –≤—ã—á–∏—Å–ª–µ–Ω–∏–µ —Å–ø–∏—Å–∫–∞.

> –ù–∞ —Å–∞–º–æ–º –¥–µ–ª–µ, –∞–ø–æ—Å—Ç—Ä–æ—Ñ - —ç—Ç–æ —Å–æ–∫—Ä–∞—â–µ–Ω–∏–µ –¥–ª—è —Å–ª–µ–¥—É—é—â–µ–π —Ñ–æ—Ä–º—ã —Å –∫–ª—é—á–µ–≤—ã–º —Å–ª–æ–≤–æ–º **quote**:  
> (quote (1 2 3 4 5)).

–°–ø–∏—Å–∫–∏ –º–æ–≥—É—Ç —Å–æ—Å—Ç–æ—è—Ç—å –∏–∑ —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –∏–º–µ—é—â–∏—Ö —Ä–∞–∑–Ω—ã–π —Ç–∏–ø. –≠—Ç–æ —Å–≤–æ–π—Å—Ç–≤–æ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è *–≥–µ—Ç–µ—Ä–æ–≥–µ–Ω–Ω–æ—Å—Ç—å—é*:

```scheme
(list "\u03BB" 42 (list #t pi))
```

–ó–Ω–∞—á–µ–Ω–∏–µ–º —ç—Ç–æ–≥–æ –≤—ã—Ä–∞–∂–µ–Ω–∏—è —è–≤–ª—è–µ—Ç—Å—è —Å–ø–∏—Å–æ–∫ *'("Œª" 42 (#t 3.141592653589793))*.

## –ü–∞—Ä—ã –∏ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–µ —É—Å—Ç—Ä–æ–π—Å—Ç–≤–æ —Å–ø–∏—Å–∫–æ–≤

–ü—Ä–∏–º–∏—Ç–∏–≤–æ–º —Å–æ—Å—Ç–∞–≤–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö –≤ —è–∑—ã–∫–µ Racket —è–≤–ª—è–µ—Ç—Å—è *–ø–∞—Ä–∞*, –∫–æ—Ç–æ—Ä–∞—è —Å–æ–∑–¥–∞—ë—Ç—Å—è —Å –ø–æ–º–æ—â—å—é —ç–ª–µ–º–µ–Ω—Ç–∞—Ä–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ **cons**:

```scheme
(cons 1 2)
```

–ï—ë —Ç–∞–∫–∂–µ –Ω–∞–∑—ã–≤–∞—é—Ç —Ç–æ—á–µ—á–Ω–æ–π –ø–∞—Ä–æ–π, –ø–æ—Ç–æ–º—É —á—Ç–æ –æ–Ω–∞ –∏–º–µ–µ—Ç —Å–ª–µ–¥—É—é—â–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ:

```scheme
'(1 . 2)
```

–ü–µ—Ä–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç –ø–∞—Ä—ã –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è *–≥–æ–ª–æ–≤–æ–π*, –∞ –≤—Ç–æ—Ä–æ–π - *—Ö–≤–æ—Å—Ç–æ–º*. –ú—ã –º–æ–∂–µ–º –ø–æ–ª—É—á–∏—Ç—å –≥–æ–ª–æ–≤—É –∏ —Ö–≤–æ—Å—Ç —Å –ø–æ–º–æ—â—å—é —ç–ª–µ–º–µ–Ω—Ç–∞—Ä–Ω—ã—Ö —Ñ—É–Ω–∫—Ü–∏–π **car** –∏ **cdr**:

```scheme
(car (cons 1 2)) ; 1 - –≥–æ–ª–æ–≤–∞ –ø–∞—Ä—ã
(cdr (cons 1 2)) ; 2 - —Ö–≤–æ—Å—Ç –ø–∞—Ä—ã
```

*–†–µ–∫—É—Ä—Å–∏–≤–Ω–æ–µ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ —Å–ø–∏—Å–∫–∞*: —Å–ø–∏—Å–æ–∫ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –ª–∏–±–æ –ø—É—Å—Ç–æ–π —Å–ø–∏—Å–æ–∫ '(), –ª–∏–±–æ –ø–∞—Ä—É, –≥–æ–ª–æ–≤–æ–π –∫–æ—Ç–æ—Ä–æ–π —è–≤–ª—è–µ—Ç—Å—è –ª—é–±–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, –∞ —Ö–≤–æ—Å—Ç–æ–º - –¥—Ä—É–≥–æ–π —Å–ø–∏—Å–æ–∫ (–ø—É—Å—Ç–æ–π –∏–ª–∏ –Ω–µ—Ç):

```scheme
(cons 1 '(2 3)) ; '(1 2 3)
```

–°–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, –º—ã –º–æ–∂–µ–º —Å–æ–∑–¥–∞—Ç—å —Å–ø–∏—Å–æ–∫ —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:

```scheme
(cons 1 (cons 2 (cons 3 null))) ; '(1 2 3)
```

> null –∏ empty —è–≤–ª—è—é—Ç—Å—è —Å–∏–Ω–æ–Ω–∏–º–∞–º–∏ –ø—É—Å—Ç–æ–≥–æ —Å–ø–∏—Å–∫–∞ '().

> –î–ª—è –ø–æ–ª—É—á–µ–Ω–∏—è –≥–æ–ª–æ–≤—ã/—Ö–≤–æ—Å—Ç–∞ —Å–ø–∏—Å–∫–∞ –º–æ–∂–Ω–æ –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è car/cdr –∏–ª–∏ –∏—Ö –∞–Ω–∞–ª–æ–≥–∞–º–∏ –¥–ª—è —Å–ø–∏—Å–∫–æ–≤ - first/rest.

## –ù–µ–∫–æ—Ç–æ—Ä—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏ –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å–æ —Å–ø–∏—Å–∫–∞–º–∏

–ü—Ä–µ–¥–∏–∫–∞—Ç list? –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç #t, –µ—Å–ª–∏ –∞—Ä–≥—É–º–µ–Ω—Ç —è–≤–ª—è–µ—Ç—Å—è —Å–ø–∏—Å–∫–æ–º:

```scheme
(list? '(1 2 3)) ; #t
```

–ü—Ä–µ–¥–∏–∫–∞—Ç null? (–∏ –µ–≥–æ —Å–∏–Ω–æ–Ω–∏–º empty?) –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç #t, –µ—Å–ª–∏ —Å–ø–∏—Å–æ–∫ —è–≤–ª—è–µ—Ç—Å—è –ø—É—Å—Ç—ã–º:

```scheme
(null? '()) ; #t
(empty? '(1 2 3)) ; #f
```

–§—É–Ω–∫—Ü–∏—è length –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —Å–ø–∏—Å–∫–∞:

```scheme

```

–§—É–Ω–∫—Ü–∏—è map –ø—Ä–∏–º–µ–Ω—è–µ—Ç—Å—è –¥–ª—è –∏—Ç–µ—Ä–∞—Ü–∏–∏ –ø–æ —Å–ø–∏—Å–∫—É:

```scheme
(map (lambda (x) (* x x))
     '(1 2 3 4 5))
```

map –ø—Ä–∏–º–µ–Ω—è–µ—Ç —Ñ—É–Ω–∫—Ü–∏—é (–ø–µ—Ä–≤—ã–π –∞—Ä–≥—É–º–µ–Ω—Ç) –∫–æ –≤—Å–µ–º —ç–ª–µ–º–µ–Ω—Ç–∞–º —Å–ø–∏—Å–∫–∞ (–≤—Ç–æ—Ä–æ–π –∞—Ä–≥—É–º–µ–Ω—Ç).  
–í –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω–æ–º –≤—ã—à–µ –ø—Ä–∏–º–µ—Ä–µ –ø–µ—Ä–≤—ã–º –∞—Ä–≥—É–º–µ–Ω—Ç–æ–º map —è–≤–ª—è–µ—Ç—Å—è –ª—è–º–±–¥–∞-—Ñ—É–Ω–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∫–≤–∞–¥—Ä–∞—Ç —á–∏—Å–ª–∞. –†–µ–∑—É–ª—å—Ç–∞—Ç–æ–º –ø—Ä–∏–º–µ–Ω–µ–Ω–∏—è —ç—Ç–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ –∫ —Å–ø–∏—Å–∫—É —á–∏—Å–µ–ª —è–≤–ª—è–µ—Ç—Å—è —Å–ø–∏—Å–æ–∫ –∫–≤–∞–¥—Ä–∞—Ç–æ–≤ –∑–∞–¥–∞–Ω–Ω—ã—Ö —á–∏—Å–µ–ª: '(1 4 9 16 25).

> map –º–æ–∂–µ—Ç –ø—Ä–∏–º–µ–Ω—è—Ç—å—Å—è –¥–ª—è –æ–±—Ö–æ–¥–∞ –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö —Å–ø–∏—Å–∫–æ–≤ –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ, –Ω–æ —ç—Ç–∏ —Å–ø–∏—Å–∫–∏ –¥–æ–ª–∂–Ω—ã –∏–º–µ—Ç—å –æ–¥–∏–Ω–∞–∫–æ–≤—É—é –¥–ª–∏–Ω—É.

–§—É–Ω–∫—Ü–∏—è filter –ø—Ä–∏–º–µ–Ω—è–µ—Ç—Å—è –¥–ª—è —Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏–∏ —Å–ø–∏—Å–∫–∞:

```scheme
(filter (lambda (x) (> x 0))
        '(-2 -1 0 1 2))
```

filter –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —Ñ—É–Ω–∫—Ü–∏—é-–ø—Ä–µ–¥–∏–∫–∞—Ç (–ø–µ—Ä–≤—ã–π –∞—Ä–≥—É–º–µ–Ω—Ç) –¥–ª—è –≤—ã–±–æ—Ä–∞ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –∑–∞–¥–∞–Ω–Ω–æ–≥–æ —Å–ø–∏—Å–∫–∞ (–≤—Ç–æ—Ä–æ–π –∞—Ä–≥—É–º–µ–Ω—Ç).  
–í –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω–æ–º –≤—ã—à–µ –ø—Ä–∏–º–µ—Ä–µ –ø–µ—Ä–≤—ã–º –∞—Ä–≥—É–º–µ–Ω—Ç–æ–º filter —è–≤–ª—è–µ—Ç—Å—è –ª—è–º–±–¥–∞-—Ñ—É–Ω–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç #t, –µ—Å–ª–∏ —á–∏—Å–ª–æ —è–≤–ª—è–µ—Ç—Å—è –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–º. –†–µ–∑—É–ª—å—Ç–∞—Ç–æ–º —Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏–∏ —è–≤–ª—è–µ—Ç—Å—è —Å–ø–∏—Å–æ–∫ '(1 2).

–§—É–Ω–∫—Ü–∏–∏ —Å–≤—ë—Ä—Ç–∫–∏ foldl (–ª–µ–≤–∞—è –∞—Å—Å–æ—Ü–∏–∞—Ç–∏–≤–Ω–æ—Å—Ç—å) –∏ foldr (–ø—Ä–∞–≤–∞—è –∞—Å—Å–æ—Ü–∏–∞—Ç–∏–≤–Ω–æ—Å—Ç—å):

```scheme

```

> –í—Ç–æ—Ä–æ–π –∞—Ä–≥—É–º–µ–Ω—Ç –≤ –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å–≤—ë—Ä—Ç–∫–∏ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è *–∞–∫–∫—É–º—É–ª—è—Ç–æ—Ä–æ–º* (–ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è, –∫–æ—Ç–æ—Ä–∞—è —Ö—Ä–∞–Ω–∏—Ç –ø—Ä–æ–º–µ–∂—É—Ç–æ—á–Ω—ã–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç).

> –ò–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ–± –æ—Å—Ç–∞–ª—å–Ω—ã—Ö —Ñ—É–Ω–∫—Ü–∏—è—Ö –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å–æ —Å–ø–∏—Å–∫–∞–º–∏ –º–æ–∂–Ω–æ –ø–æ—á–µ—Ä–ø–Ω—É—Ç—å –≤ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–µ–º [—Ä–∞–∑–¥–µ–ª–µ –¥–æ–∫—É–º–µ–Ω—Ç–∞—Ü–∏–∏](https://docs.racket-lang.org/reference/pairs.html).
